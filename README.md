## 🌐 Personal Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, projects, and journey in web development. It's a static site built using pure HTML and CSS, designed with clarity and simplicity in mind.

### ✨ Features

* Clean and minimal user interface
* Smooth transitions and subtle animations
* Sections like About Me, Skills, Projects, Education, and Contact
* Easy navigation through anchor links
* Organized file structure for assets (images, icons, etc.)

### 📁 Technologies Used

* HTML5
* CSS3 (including transitions and keyframe animations)
* No external libraries or frameworks

### 🚀 Deployment

This project is best hosted on static site platforms like **Netlify**, **GitHub Pages**, or **Vercel**.
All paths are relative for easy drag-and-drop deployment.

### ⚠️ Important Note

> 📱 **This website is currently optimized for desktop only and does not support mobile responsiveness or portability.** Future updates may include media queries for improved mobile experience.
